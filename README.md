应用程序构建包的脚本和补丁。

关于 Termux 包管理的快速说明可在 [包管理](https://github.com/termux/termux-packages/wiki/Package-Management) 中获取。它还包含如何在运行 `apt` 或 `pkg` 命令时修复 **`repository is under maintenance or down`** 错误的信息。
